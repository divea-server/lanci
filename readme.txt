== Description ==

Habilita os Caches Redis e Nginx Proxy Cache da sua hospedagem para otimizar o seu Wordpress.

Limpa automaticamente os caches quando há uma alteração ou publicação de posts, páginas, menus, widgets e configurações.

  * Redis 2.0.15
  * Nginx Cache 1.0.5

= Requisitos =

  * Adicionar no arquivo wp-config.php a constante: define("LANCI_SERVER_ALIAS" , "<alias-da-sua-instalação>");
